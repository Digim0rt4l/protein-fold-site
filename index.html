Each work unit runs for a long stretch on purpose, so your device stays useful without hammering the shared server. Pause anytime.

This page is running a simplified, coarse-grained conformational search on amyloid-beta 1-42, the peptide known as PDB entry 1IYT, whose aggregation is central to the amyloid hypothesis of Alzheimer's disease.

Every visitor's device is handed a small work unit, a stretch of the backbone, and runs a long simulated-annealing search on it in a background thread. Results are sent to a server function, independently re-scored, and merged into one shared structure stored in this project's GitHub repository, only when they genuinely lower the total energy.

Please calibrate your expectations. This is an educational, citizen-science-style demo of distributed computing and coarse-grained energy minimization, built for learning and engagement. It is not a production molecular-dynamics engine like Folding at home, and it is not a substitute for tools like AlphaFold or Rosetta. It will not produce new, publishable Alzheimer's science on its own, but it is a genuine, functioning distributed system you can watch work in real time.
